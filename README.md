# TradeX 
AI-Driven Multi-Asset Trading & Learning System

TradeX is an advanced trading platform that combines institutional mimicry, sentiment analysis, and ethical scoring to provide a robust environment for algorithmic trading.

## Key Features
- **Multi-Asset Ingestion**: Stocks, Forex, Crypto, and Bonds.
- **Institutional Mimicry**: Follow styles of NBIM, BlackRock, and Renaissance.
- **Ethical Filtering**: Corruption Index and ESG scoring for all assets.
- **AI Coach Network**: Specialized bots for Risk, Growth, Sentiment, and Ethics.
- **Guardrails**: Automated financial safety limits (drawdown/position sizing).
- **Docker Ready**: Fully containerized for easy deployment.

## Quick Start
1. **Train the AI**: `python train_system.py`
2. **View Status**: `python status_report.py`
3. **Run Trading**: `python main.py`
